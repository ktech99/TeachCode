Our Aim

Today programming is taught in nearly every school. But what students are taught is how to code, not how to think. In other words, the computer curriculum in schools teaches students only basic syntax but not how to apply this knowledge. We aim to help students learn how to think while coding and teach them how to crack problem statements.

What Is Programming?

Programming is how you interact with a computer by giving it certain instructions to perform a particular task.
Every time you do a particular task on the computer or any device, a specific set of underlying code is executed which you cannot see.
Through programming you can create many amazing things, such as: a website, a game, a house automating system… The possibilities are endless.

Not only do these techniques help a student in understanding how the world works, but also gives them a deeper understanding of computer code in a fun and entertaining manner, which would promote them to pursue computer science in the future.

Method

To learn to solve problems we will be using the three step approach to problem solving

Analyse

Conceptualise

Execute

Analyse

The first step for you would be to read the question carefully and understand your problem statement.
A good practice would be to write down on a sheet of paper what is expected of you.
For e.g. you are asked to calculate the sum of the Fibbonacci Series. You should note down sum=1+2+3+5+8….
A good place to practise this would be codechef.org

Conceptualise

For a few minutes, forget about classes or methods. Forget about what functions to use. Just conceptualise the way the program will work to accomplish the desired goal. Make a flow chart tracing the flow of your program. Dry run the flow of your program by putting test values and seeing if it gives the desired output. Don't be afraid to start over again.

Execute

The last part of your program is where you write down your program on the computer. Here you convert the flowchart you just made into working code.

Visualisation

One ability to become a good programmer is the ability to visualise.It is extremely important to imagine or visualise what your code is going to do on the screen.
A good practice for new programmers is to close your eyes and think of a programming statement, then think about how it will affect the output on the screen. Then go back to the program and add a line, think back to how it will affect your output.

If-Else

If-Else is a conditional statement which checks if the condition passed is true or false.

For eg: The code on the left takes a variable ‘a’ as input.
It then checks if a>10.
If this condition is true, the program prints ”Value of a is greater than 10”.
If this condition is false, the program skips the IF block and moves on to the Else block.
Once it encounters the Else block it prints “Value of a is smaller than 10”.

Imp: Once the condition is true, the program will automatically skip the Else block.

for(x=0; x<10; x++)

print(x is increasing)

For-Loop

The for loop is an iterative function. This means that the statement within the loop will keep getting executed till the specified condition is satisfied.

For eg: The code on the left first initialises x to 0.
Then it checks if the condition x<10 is satisfied.
If the condition is satisfied, the loop is executed and the program prints, “x is increasing”.
If the condition is not satified, the loop does not run.
After the loop runs x gets increased by a value of 1, due to the increment operator, x++.
Now x has the value of 1 therefore it satisfies the case x<10 and the loop will run again.
This keeps continuing till x=10.
Once x=10, the condition is not satisfied and the loop does not run any more.

This will be much more clear when you practice this a couple of times. A good idea would be to try making patterns using loops.

i=0

While(i<10)
{
Print(While loop is running)

i++
}

While-Loop

The While loop is very much similar to the for loop.
The only differences are that you cannot initialise the value of a variable in the start of a while loop, and the increment or decrement function has to be inside the loop.
Otherwise this loop will run exactly like the previous one.

i=0

do
{
print(“I will run atleast once”)
i++;
} while(i<10)

Do While-Loop

Do while loop, unlike the other loops is an exit controlled loop.
What this means is that the loop will run atleast once regardless if the condition is satisfied or not.
After running once, the program will check if the statement condition is satisfied.
If it is satisfied, the loop runs again and if it is not, the loop is exited.
