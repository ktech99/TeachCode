Nested If-Else

Nested functions are functions which are placed inside another function.
This is sometimes necessary for certain programs, but it makes them slightly more complicated.
In this type of function, first the compiler will check if the outer function is satisfied, then only will it be able to check the inner function.
Let us see this with the example given.

Lets say the day is saturday.
Therefore the outer if case is true and the control will enter the first If block.
Now the compiler encounters the second If statement.
Now if the time=4 the program enters the second if block and prints “It is time to study”.
If the time Is not=4 the program enters the inner else block instead and prints “It is time to party”.

Now lets take the case that the day is monday.
As day is not=saturday, The control will not enter the first if block, but Instead it will enter the else block and prints “It is not a saturday”.

Note: A good practice while using nested functions would be to use indentation.
