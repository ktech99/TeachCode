Nested Loops

Just Like nested If else, there are also nested loops.
In this there is one loop inside another loop. This might look intimidating at first, but when you understand how it works it’s quite simple.
In nested loops, the control checks the outer loop condition first.
The control enters the loop and runs the print statement.
Now it checks if the inner loop condition is satisfied.
The inner loop runs normally like it would have.
Now once the inner loop stops running, the control goes to the outer loop again and now i is increased by 1.
Every time the outer loop increases, the inner loop is reset to 0 only in the case of for loop.

The output of this program would be:
0
1
1
0
1
1

Try playing with the code and changing it to get more clarity on this topic.
